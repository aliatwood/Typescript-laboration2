Laboration 2 - Objektorienterad programmering

## Beskrivning
En Todo-applikation byggd med TypeScript där användaren kan lägga till, markera och ta bort uppgifter. 
Uppgifterna har prioritet (1–3) och sparas i LocalStorage för att behålla data mellan sidladdningar.
